Stream Chrome Extension

This project was started as a midterm project from the Jan 2015 web development cohort at Lighthouse Labs in Vancouver, BC. It started out as a simple Sinatra application running on a local machine, and was later expanded on by the team leader into a full-featured app, incorporating the use of JQuery and Ajax to handle CORS specs.
